# Product Listing Page (PLP) - Frontend Development Project

## 1. Objective

The goal of this project is to create a fully functional **Product Listing Page (PLP)** 
This page simulates an e-commerce listing interface, including:

- Header with navigation and search  
- Sidebar filters for refining products  
- Product grid with cards (images, names, prices)  
- Footer section  

The project demonstrates front-end skills such as:

- Translating UI designs into responsive HTML, CSS, and React.js code  
- Ensuring responsiveness (desktop, tablet, mobile)  
- Optimizing SEO (semantic HTML, meta tags, alt attributes)  
- Optionally integrating mock API data (e.g., Fake Store API)  
- Deploying a live version using GitHub and Netlify  

By completion, the evaluator should find a **responsive, clean-coded, and optimized** product listing page with strong attention to UI/UX detail and minimal DOM complexity.

---

## 2. Technical Requirements

### Software & Tools
- **Code Editor**: Visual Studio Code  
- **Version Control**: Git + GitHub account  
- **Node.js and npm**: Version 14+  
- **Browser**: Google Chrome or Firefox  
- **Deployment**: [Netlify](https://www.netlify.com/) (free tier)  
- **Optional Mock API**: [Fake Store API](https://fakestoreapi.com/)  

### Libraries / Packages
- **React.js (via Create React App)**  
- **Axios** *(optional)* for API calls  
- No heavy CSS frameworks (Bootstrap etc.) — use **pure CSS / Tailwind**  
- Minimal dependencies  

### Skills Required
- Basic understanding of HTML, CSS, JavaScript, and React fundamentals
